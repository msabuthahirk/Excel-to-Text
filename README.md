It is a python3 gui app to extract a single column from Excel(*.xls,*.xlsx) to Text(.txt)


Packages needed:
	xlrd
	tkinter

For the installation of above packages:
just copy these codes to your terminal
<code>
pip3 install --upgrade xlrd
</code>

for execution:
just run
<code>
python gui.py
</code>
or
<code>
python3 hui.py
</code>
